# Breeze Integrations

This module includes Breeze integrations with Swissup modules.

## Installation

```bash
composer require swissup/module-breeze-integrations
bin/magento setup:upgrade --safe-mode=1
```

## Available integrations

 -  Swissup_Ajaxpro
 -  Swissup_Pagespeed
 -  Swissup_ProLabels
